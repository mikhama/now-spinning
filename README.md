# now-spinning
Hobby project with Raspberry Pi.


## Utils

To read NFC tag:
```bash
python -m exp.nfc.nfc_read
```

To write NFC tag:
```bash
python -m exp.nfc.nfc_write
```

To diagnose NFC card type:
```bash
python -m exp.nfc.nfc_diagnose
```

To debug PN532 module:
```bash
python -m exp.nfc.nfc_debug
```
