Design System Reference

My Musical Journey

Complete visual reference for adapting this design to a Raspberry Pi embedded UI. Vintage newspaper aesthetic with warm paper tones, vinyl-record motifs, and strong typographic hierarchy.

Typography

Three font families create a strong hierarchy. Display type uses Gloock, body uses Fraunces, and all metadata/utility text uses DM Mono in uppercase with wide letter-spacing.

Gloock — Display / Headings

letter-spacing: -0.04em (large sizes) to -0.02em (medium)
line-height: 0.85–1.0

Background & Texture

The page uses a layered background system: solid paper color → radial gradient warm spots → SVG noise grain overlay.

Background Layers (bottom to top)
Layer 1: Solid Base
background: var(--paper) → #efe6d4
Layer 2: Warm Gradients
Two radial-gradients: amber tint top-left (8% opacity), dark tint bottom-right (5% opacity)
Layer 3: Paper Grain
SVG feTurbulence fractalNoise, baseFreq 0.9, 3 octaves, opacity 0.35, mix-blend-mode: multiply. Fixed position, covers viewport.

Vinyl Record Element

The record peeks out from behind the album sleeve. It spins on hover. Key visual specs below.

Record Anatomy
Disc Surface: repeating-radial-gradient (groove simulation) + base radial gradient
Groove overlay: inset 8%, repeating-radial-gradient 3px interval, 3% white opacity
Center label: 36% diameter, amber radial gradient, 8% center hole
Animation: @keyframes spin { 0%→360° }, 8s linear infinite, paused by default
Hover slide: translateX(38%) rotate(6deg), 0.8s cubic-bezier(.2,.8,.2,1)
Sleeve 3D tilt on hover: rotateY(-4deg) rotateX(2deg), perspective: 1200px
Sleeve corner marks: 22×22px, border 1px solid rgba(255,255,255,0.35), top-left + bottom-right

Animations & Interactions

Minimal, tasteful motion. Scroll-driven entry reveals and hover interactions.

Entry fade-in: opacity 0→1, translateY(30px→0), 1s cubic-bezier(.2,.8,.2,1)
  animation-timeline: view(), range: entry 0%–100%
Sleeve hover: rotateY(-4deg) rotateX(2deg), 0.6s cubic-bezier(.2,.8,.2,1)
Record hover: translateX(38%) rotate(6deg), 0.8s cubic-bezier(.2,.8,.2,1) + spin starts
Link hover: background/color invert, 0.25s ease
Timeline active: border-right-color + dot scale(1.5) + amber color, 0.25s ease
Sparked card hover: border-color → amber-deep, bg → rgba(184,97,43,0.06), 0.2s ease
prefers-reduced-motion: all animations disabled, static layout
